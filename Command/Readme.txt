Command is behavioral design pattern that converts requests or simple operations into objects.

The conversion allows deferred or remote execution of commands, storing command history, etc.

/*-----------------------------------------*/

Command (lệnh) là mẫu thiết kế hành vi chuyển đổi các yêu cầu hoặc hoạt động đơn giản thành các đối tượng.

Việc chuyển đổi cho phép thực hiện lệnh hoãn lại hoặc từ xa, lưu trữ lịch sử lệnh, v.v.
